

# AI Ethics: MidJourney – Ethical & Unethical Uses

## 📌 Overview
This project analyzes the ethical implications of generative AI tools, focusing on MidJourney.

## 🧠 Objective
To evaluate both the positive and negative impacts of AI-generated art.

## ⚖️ Topics Covered
- Ethical uses (education, creativity, communication)
- Unethical uses (privacy, misinformation, job displacement)
- Impact on artists and journalism

## 🎯 Outcome
Provided a balanced perspective on the societal impact of generative AI tools.

## 📚 Type
Academic / Ethics Analysis
